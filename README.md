## Description

A [Nest](https://github.com/nestjs/nest) application to handle lessons and students entities, using graphql, 
mongoDB and TypeORM.

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn start

# watch mode
$ yarn start:dev

# production mode
$ yarn start:prod
```

## Test

```bash
# unit tests
$ yarn test

# e2e tests
$ yarn test:e2e
```
